# Accoutrement

A Live music and SFX mixer for Discord. Control a discord bot with our UI and mix audio for entertainment.

In particular, I made this application for GM'ing TTRPG (Table Top Roleplaying Game) sessions. 

## Features
* Play music
* Play multiple sound effects concurrently
* Load media files from disk
* Control bot voice connection

## Planned Features
* Playlist files
* Play from youtube links
